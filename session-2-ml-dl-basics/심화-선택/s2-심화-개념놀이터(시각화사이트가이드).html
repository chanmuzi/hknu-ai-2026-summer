① GRADIENT DESCENT

경사하강법 — 손실이 낮은 쪽으로 내려간다

모델은 지금 위치의 기울기(경사)를 보고 손실이 줄어드는 쪽으로 한 걸음씩 내려갑니다. 학습률이 걸음 폭이에요 — 너무 크면 골짜기를 지나쳐 튕겨 나갑니다.

이 사이트에서 해볼 것
  1. 학습률(learning rate) 값을 작게 → 크게 바꿔 가며 실행해 보세요.
  2. 작으면 천천히 또박또박, 적당하면 빠르게 바닥으로 내려갑니다.
  3. 너무 키우면 경로가 골짜기를 지나쳐 튕기거나 발산하는 걸 봅니다.
Gradient Descent Visualizer ↗ UCLA ACM 버전 ↗

2차시 연결: 경사를 따라 손실을 줄인다 · 학습률이 학습의 성패를 가른다

② K-MEANS CLUSTERING

K-means — 중심을 옮기며 무리를 묶는다

정답(label)이 없는 점들을 비슷한 것끼리 묶습니다. 배정(각 점 → 가장 가까운 중심)과 이동(중심 → 무리의 평균)을 번갈아 반복하면 중심이 더 안 움직일 때 수렴합니다.

이 사이트에서 해볼 것
  1. 위쪽에서 데이터 모양을 고릅니다(예: Gaussian Mixture).
  2. 중심(centroid)을 직접 클릭해 배치하거나 초기화 방법을 고릅니다.
  3. Update Centroids / Reassign Points를 번갈아 누르며, 점 색이 바뀌고 중심이 평균으로 이동해 수렴하는 걸 봅니다.
Visualizing K-Means (Naftali Harris) ↗

2차시 연결: 군집화는 중심을 옮기며 묶는다 · 정답 없는 비지도학습

③ NEURAL NETWORK

진짜 신경망 해보기 — TensorFlow Playground

버튼이 많아 막막하지만, 핵심은 하나입니다: 신경망이 점들을 갈라내는 '경계'를 스스로 학습하는 걸 지켜보는 것.

이 사이트에서 해볼 것 (5단계)
  1. playground.tensorflow.org 를 엽니다. (가입·설치 없음)
  2. 왼쪽 DATA에서 점 모양을 하나 고릅니다. 처음엔 쉬운 두 덩어리 추천.
  3. 가운데 위 ▶ (재생) 버튼을 누릅니다. 신경망이 학습을 시작합니다.
  4. 오른쪽 그림에서 파랑·주황 경계가 또렷해지며 점을 갈라내는 걸 봅니다. 왼쪽 위 Loss가 줄수록 잘 배우는 중.
  5. HIDDEN LAYERS의 +로 층·뉴런을 늘리고, 데이터를 소용돌이로 바꿔 다시 ▶. 층이 깊을수록 복잡한 경계를 학습합니다.
TensorFlow Playground 열기 ↗

2차시 연결: 층을 깊게 쌓을수록 더 복잡한 표현 · 색 경계 = 학습된 분류 경계
